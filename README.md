# 📝 스마트노트 (SmartNote)

AI 요약 ✨ 음성 인식 🎙 이미지 텍스트 분석 📸

- ✏️ 노트 작성
- 📸 이미지 텍스트 인식 (OCR)
- 🎤 음성 인식 (STT)
- 🤖 AI 요약
- 📝 시험 문제 생성
